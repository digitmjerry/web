01

為什麼需要這些設定？

電腦是存取公司資料、個人帳戶的重要入口。一旦未設定密碼或未啟用自動上鎖，任何走近電腦的人都可能在您離開座位的片刻，存取您的信箱、文件、系統，甚至安裝惡意軟體。

⚠ 常見的風險情境
離開座位去倒水、上廁所、開會，就算只有 3 分鐘，未上鎖的電腦可能已遭人翻閱資料或植入惡意程式。
✅ 本 SOP 目標
完成設定後，當您離開座位超過指定時間，電腦將自動鎖定，需要輸入密碼才能繼續使用，大幅降低資料外洩風險。

02

強密碼原則

密碼是第一道防線，請務必遵守以下原則：

🔢
至少 12 個字元
字元愈長，破解所需時間呈指數增長
🔡
大小寫混用
同時包含大寫字母 A–Z 與小寫字母 a–z
🔣
加入符號與數字
如 !@#$%^&* 與 0–9
🎲
無規律可循
避免鍵盤序列或生日等可預測資訊
🚫
禁用：123456
全球最常見密碼，幾秒即被破解
🚫
禁用：姓名 / 生日
社交工程攻擊首選的猜測目標
🚫
禁用：重複使用
一個帳號洩漏，所有帳號全部淪陷
🔐
使用密碼管理器
Bitwarden、1Password 等工具安全儲存
💡 好用技巧：密碼短語（Passphrase）
挑選 4 個不相關的詞組成句子，例如 椅子-火箭-藍莓-地圖!（加上符號），長度夠長又好記，同時具有高強度。

03

Windows：設定登入密碼

🪟 Windows 10 / 11 — 設定登入密碼
  1. 點擊左下角「開始」按鈕，選擇「⚙ 設定」（齒輪圖示）。
  2. 進入「帳戶」→「登入選項」。
  3. 在「密碼」區塊，點選「新增」（若已有密碼則顯示「變更」）。
  4. 輸入新密碼，再次確認，並填入「密碼提示」（請勿直接寫出密碼），完成後按「下一步」→「完成」。
  5. 快速驗證：按下 Win + L 鎖定電腦，再輸入密碼解鎖，確認設定成功。
💡 Windows Hello（進階）
若裝置支援，可在「登入選項」中額外設定指紋或臉部辨識，作為密碼之外的快速解鎖方式，但主密碼仍必須設定。

04

macOS：設定登入密碼

🍎 macOS Ventura / Sonoma — 設定登入密碼
  1. 點擊左上角「🍎」Apple 選單，選擇「系統設定」（Ventura 以後）。
  2. 在左側點選您的 Apple ID 名稱（最頂端），或直接搜尋「密碼」。
  3. 前往「一般」→「登入密碼」，點選「更改密碼⋯」。
  4. 若目前無密碼，直接輸入新密碼兩次並確認；若已有密碼，先輸入舊密碼再設定新密碼。
  5. 快速驗證：按下 Control + Command + Q 鎖定螢幕，再輸入密碼解鎖確認成功。
💡 Touch ID（進階）
macOS 支援 Touch ID 指紋解鎖，可在「系統設定 → Touch ID 與密碼」中啟用。Touch ID 為密碼的補充，無法取代密碼設定。

Windows：設定自動螢幕上鎖

使用場景 建議鎖定時間 評等
開放式辦公室 / 共用空間1–3 分鐘★ 強烈建議
個人辦公室（上鎖）5 分鐘✓ 推薦
居家工作5–10 分鐘✓ 推薦
30 分鐘以上—✗ 不建議
從不鎖定—✗ 嚴禁
🪟 Windows 10 / 11 — 螢幕保護程式自動上鎖（方法 A）
  1. 在桌面空白處按右鍵 →「個人化」。
  2. 點選左側「鎖定畫面」→ 拉到底部點選「螢幕保護程式設定」。
  3. 選擇任一螢幕保護程式（如「空白」），設定等待時間（建議 3–5 分鐘）。
  4. 勾選「繼續顯示登入畫面」，按「套用」→「確定」。

06

macOS：設定自動螢幕上鎖

🍎 macOS Ventura / Sonoma — 自動上鎖設定
  1. 開啟「🍎 系統設定」→「隱私權與安全性」。
  2. 找到「需要密碼」選項，下拉選單選擇「立即」或「5 分鐘後」（建議選立即或 1 分鐘）。
  3. 再前往「螢幕保護程式（System Settings → Screen Saver）」，設定閒置多久後啟動（建議 3 分鐘）。
  4. 確認「需要密碼」設為「立即」，螢幕保護程式一啟動即需密碼解鎖。
  5. 設定熱角（Hot Corners）：「系統設定 → 桌面與 Dock → 熱角⋯」，將某角設為「鎖定螢幕」，離座前滑鼠移到該角即可立即上鎖。
⚡ 快速手動上鎖
任何時候按下 Control + Command + Q 可立即鎖定螢幕，無需等待計時器。養成離座前按此快速鍵的習慣！
§
07

🚫 禁止將密碼貼在明顯處

將密碼寫在便條紙、貼在螢幕邊框、鍵盤底部，或夾在桌上文件夾中，是最常見、也是最危險的資安漏洞之一。

❌ 絕對禁止

密碼：Abc@12345!
帳號：[email protected]

這樣的便條紙讓密碼形同虛設

✅ 正確做法

🔐 使用密碼管理器
Bitwarden / 1Password / KeePass
加密儲存，隨時查閱，不怕遺忘

密碼管理器是最安全的記錄方式

⚠ 常見危險行為清單
  • 密碼寫在螢幕邊框或底座的便條紙
  • 密碼存在手機相簿的截圖或照片中
  • 密碼寫在桌曆或工作日誌的顯眼位置
  • 以「password.txt」或「帳號密碼.docx」等檔名明文儲存在桌面
  • 透過 LINE / Email 傳送密碼給自己（訊息可能遭攔截）

08

✅ 自我檢核清單

請完成以下所有項目，並點擊勾選確認。所有項目完成後即代表您已達到基本資訊安全要求。

  • 我的電腦帳號已設定強密碼（12 字元以上，含大小寫、數字、符號）
  • 密碼未使用姓名、生日或連續數字等可預測資訊
  • Windows 已設定螢幕保護程式並勾選「繼續顯示登入畫面」（Windows 用戶）
  • macOS 已設定「需要密碼」為立即或 1 分鐘以內（Mac 用戶）
  • 自動上鎖時間設定為 5 分鐘以內
  • 我已記住手動鎖定快速鍵（Win+L / Ctrl+Cmd+Q）並養成離座前鎖定的習慣
  • 密碼未寫在便條紙或任何可被他人看見的地方
  • 密碼未以明文存在桌面文件或手機相簿中
  • 我已考慮使用密碼管理器來安全儲存所有密碼
  • 我了解若密碼外洩應立即向 IT 部門回報並更換密碼
🎉 恭喜！所有項目已完成
您已完成基本資訊安全設定，請定期（每 90 天）更換密碼，並持續關注MIS發布的最新資安公告。
